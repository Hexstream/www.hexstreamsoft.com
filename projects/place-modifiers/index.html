place-modifiers essentially gives access to hundreds of modify-macros through one single macro: MODIFY.

Nearly-complete documentation on Github. I will complete this and publish it in HTML here soon.
(Update 22 october 2012: Well, not sure about "soon" anymore. My apologies!)

To use place-modifiers, simply (:import-from #:place-modifiers #:modify). Don't (:use)!

(ql:quickload "place-modifiers") (September 2012 quicklisp dist (or later) required, run (ql:update-dist "quicklisp") first if necessary.)
